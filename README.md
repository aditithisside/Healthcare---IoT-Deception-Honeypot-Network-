# Healthcare IoT Deception Honeypot Network

## Project Overview
This project simulates a Healthcare IoT environment using a Cowrie Honeypot to capture and analyze attacker behavior.

## Tools Used
- Kali Linux
- Docker
- Cowrie Honeypot
- Loki
- Promtail
- Grafana
- GitHub

## Features
- SSH Honeypot
- Login Attempt Monitoring
- Malware Download Attempt Detection
- Dashboard Visualization
